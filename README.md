## 🎨 Visual Identity & Premium Design System
The application features a modern, tech-focused dark glassmorphism aesthetic:
- **Core Theme**: Deep space navy and charcoal base (`#05070f`).
- **Accent Elements**: Electric Blue (`#00d2ff`) representing technical automation systems and Gold/Amber (`#ffc107`) representing high-value monetization and compounding assets.
- **Glassmorphism Grid**: Translucent layout cards with `backdrop-filter: blur(16px)` and extremely subtle light-border overlays that glow dynamically.
- **Dynamic Interactions**: Micro-scale animations on badge hover, custom smooth scrolling, active navigation highlights, and interactive radial glow coordinates tracking mouse movement.

## 🚀 How to Run & View Locally
Since the application is designed to be highly performant, lightweight, and self-contained, there are **no heavy Node modules, external dependencies, or complex build pipelines to install**.
### Quick Start
1. Navigate to the folder containing `index.html`.
2. **Double-click** the `index.html` file to open it directly in Google Chrome, Microsoft Edge, Safari, or Firefox.
### Local Development Server (Recommended)
For full features such as smooth file path mapping and live-reloads during modifications:
- **VS Code**: Install the **Live Server** extension, right-click `index.html`, and select **"Open with Live Server"**.
- **Python**: Run the following command in your terminal from this directory:
  ```bash
  python -m http.server 8000
  ```
  Then open `http://localhost:8000` in your web browser.
